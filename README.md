# Adds creative like flight with a chicken aftertaste.

# Recipe

Ring can be crafted using:
 2 elytras
 4 nether stars
 1 copper ring (Amethyst imbuement mod dependent)
 2 corrupted chicken essence (not implemented yet)  use Diamond blocks instead.
